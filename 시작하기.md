# WorkFair 미리보기 실행 가이드 🚀

## 빠른 시작 (5분)

### 1️⃣ 프론트엔드 패키지 설치

터미널을 열고 프로젝트 폴더에서:

```bash
cd frontend
npm install
```

⏰ 설치 시간: 약 2-3분

---

### 2️⃣ 백엔드 설정

**새 터미널 창**을 열어주세요:

```bash
cd backend

# 가상환경 생성
python -m venv venv

# 가상환경 활성화
source venv/bin/activate  # Mac/Linux 사용자
# 또는
venv\Scripts\activate     # Windows 사용자

# 패키지 설치
pip install -r requirements.txt

# 데이터베이스 생성 및 테스트 데이터 추가
python -m app.seed
```

✅ 성공하면 이렇게 표시됩니다:
```
✅ Database seeded successfully!

Test accounts:
  Jobseeker: seeker1@example.com / password123
  Employer: employer1@example.com / password123
```

⏰ 설치 시간: 약 2분

---

### 3️⃣ 백엔드 서버 실행

같은 터미널에서 (venv가 활성화된 상태):

```bash
uvicorn app.main:app --reload
```

✅ 서버가 실행되면:
- 백엔드 API: **http://localhost:8000**
- API 문서: **http://localhost:8000/docs** 

⚠️ **이 터미널 창은 계속 열어두세요!**

---

### 4️⃣ 프론트엔드 서버 실행

**또 다른 새 터미널 창**을 열어서:

```bash
cd frontend
npm run dev
```

✅ 서버가 실행되면:
- 프론트엔드: **http://localhost:5173**

⚠️ **이 터미널 창도 계속 열어두세요!**

---

### 5️⃣ 브라우저에서 열기

웹 브라우저에서 **http://localhost:5173** 을 엽니다.

## 📱 모바일 뷰로 보기 (필수!)

WorkFair는 **모바일 우선**으로 디자인되었습니다. 
**375px 너비**로 보는 것이 가장 좋습니다!

### Chrome 개발자 도구 사용법:

1. **F12** 키를 누릅니다
2. **Ctrl+Shift+M** (Mac: Cmd+Shift+M) 를 누릅니다
3. 상단 드롭다운에서 **"iPhone SE"** 선택
   - 또는 직접 **375 x 667** 로 설정

### 다른 브라우저:

- **Firefox**: F12 → Ctrl+Shift+M → 375px 설정
- **Safari**: Develop → Enter Responsive Design Mode → 375px
- **Edge**: Chrome과 동일

## 🎯 확인할 페이지들

미리보기에서 확인해보세요:

### 1. 홈 화면 (`/jobseeker/home`)
- 검색바
- 추천 필터 칩 (TOPIK 2급, 주말, 비자-C-4, 종로구)
- 지원 현황 카드 (그라데이션)
- AI 맞춤 공고 (가로 스크롤)
- 학습 진도 카드
- 빠른 메뉴 그리드
- 새로운 공고 (2열 그리드)
- 생활 꿀팁 가이드 카드

### 2. 공고 상세 (`/job/job-1`)
- 가게 이름 + 별점
- 사장님의 한마디 (녹색 배너)
- 근무 조건 표
- 모집 조건 (D-8 배지)
- 하단 고정 버튼 (채팅/전화/지원하기)

### 3. 인재 필터 (`/filters/talent`)
- 언어 능력 필터
- 거주지 필터
- 경력사항 필터
- 근무 조건 필터
- 선택 시 민트색으로 변경되는 칩
- 하단 "적용하기" 버튼

### 4. 공고에 지원하기
1. 공고 카드 클릭
2. "지원하기" 버튼 클릭
3. 성공 토스트 알림 확인
4. 같은 공고에 다시 지원하면 "이미 지원한 공고입니다" 알림

## 🔧 문제 해결

### 백엔드가 시작되지 않으면:

```bash
# 포트 8000이 사용 중인지 확인
lsof -i :8000  # Mac/Linux
netstat -ano | findstr :8000  # Windows

# 프로세스 종료하고 다시 시작
```

### 프론트엔드가 시작되지 않으면:

```bash
# node_modules 삭제하고 재설치
rm -rf node_modules package-lock.json
npm install
```

### 데이터가 보이지 않으면:

```bash
# 데이터베이스 재생성
cd backend
rm workfair.db  # 기존 DB 삭제
python -m app.seed  # 재생성
```

### CORS 에러가 발생하면:

- 백엔드 서버가 먼저 실행되었는지 확인
- 프론트엔드가 http://localhost:5173 에서 실행되는지 확인
