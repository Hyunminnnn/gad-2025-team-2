# WorkFair 일정 공유 기능 가이드

## 🎯 개요

Crewmon 스타일의 일정 관리 및 공유 기능이 구현되었습니다. 고용주와 구직자가 일정을 쉽게 공유하고 관리할 수 있습니다.

### 주요 기능

- ✅ **구직자**: 내 근무 일정 확인
- ✅ **고용주**: 직원 목록 → 직원별 일정 관리
- ✅ Crewmon 스타일 깔끔한 UI
- ✅ 월별 캘린더 뷰
- ✅ 근무 통계 (총 근무, 완료, 지각, 결근)
- ✅ 색상 코딩 (완료: 초록, 예정: 파랑, 지각: 노랑, 결근: 빨강)
- ✅ 다가오는 근무 목록

## 📁 구조

### 타입 정의

```typescript
// types/schedule.ts
- DayStat: 날짜별 근무 통계
- Shift: 개별 근무 정보
- Employee: 직원 정보
- MonthSummary: 월별 요약 통계
- ScheduleMonth: 월별 일정 데이터
```

### 공용 컴포넌트

```
src/components/schedule/
├── MonthPicker.tsx        # 년/월 선택기
├── CalendarGrid.tsx       # 달력 그리드
└── ShiftCard.tsx          # 근무 카드
```

### 페이지

**고용주용:**
```
/employer/schedule                  # 직원 목록
/employer/schedule/:userId          # 특정 직원의 공유 일정
```

**구직자용:**
```
/jobseeker/schedule                 # 내 일정
```

## 🚀 사용 방법

### 1. 마이페이지에서 접근

```
마이페이지 → "일정 관리" 버튼 클릭
```

- **구직자 모드**: 자신의 근무 일정으로 이동
- **고용주 모드**: 직원 목록으로 이동

### 2. 고용주 워크플로우

```
1. 마이페이지 → 일정 관리
2. 직원 목록에서 직원 선택
3. 해당 직원의 공유 일정 확인
4. "+ 일정 추가" 버튼으로 새 근무 추가
```

### 3. 구직자 워크플로우

```
1. 마이페이지 → 일정 관리
2. 월별 캘린더에서 일정 확인
3. 다가오는 근무 목록 확인
```

## 🎨 UI 특징

### Crewmon 스타일 디자인

- **색상 팔레트**:
  - 기본: 흰색 배경, 연한 회색 표면
  - 주 액센트: 민트 색상 (#61D3A9)
  - 완료: 초록색 (#34D399)
  - 예정: 파랑색
  - 지각: 노랑색
  - 결근: 빨강색

- **둥근 모서리**:
  - 카드: 24px (`rounded-2xl`)
  - 칩: 12-16px (`rounded-xl`, `rounded-lg`)

- **그림자**: 부드러운 `shadow-sm`, `shadow-md`

- **간격**: 8px 그리드 시스템

### 캘린더 그리드

```
┌─────────────────────────────────┐
│  일  월  화  수  목  금  토       │
├─────────────────────────────────┤
│  1   2   3   4   5   6   7      │
│     [4시간30분] [+15m]           │
│                                  │
│  8   9   10  11  12  13  14     │
│          [-30m]      [결근]      │
└─────────────────────────────────┘
```

### 상태 표시

| 상태 | 색상 | 예시 |
|------|------|------|
| 완료 | 초록 | `4시간 30분` |
| 예정 | 파랑 | `5시간` |
| 지각 | 노랑 | `+15분` |
| 조퇴 | 회색 | `-30분` |
| 결근 | 빨강 | `결근` |

## 🔧 API 엔드포인트 (준비됨)

### 고용주용

```
GET /api/schedule/employees
→ 현재 고용중인 직원 목록

GET /api/schedule/shared/:userId?year=2024&month=11
→ 특정 직원과 공유하는 월별 일정

POST /api/schedule/shift
→ 새 근무 일정 추가
body: {
  userId: string,
  workplaceId: string,
  startISO: string,
  endISO: string,
  notes?: string
}
```

### 구직자용

```
GET /api/schedule/my?year=2024&month=11
→ 내 월별 일정

GET /api/schedule/upcoming?limit=10
→ 다가오는 근무 목록
```

## 💡 주요 기능 상세

### 1. MonthPicker (월 선택기)

```tsx
<MonthPicker 
  value={{ year: 2024, month: 11 }} 
  onChange={(year, month) => {}}
/>
```

- 이전/다음 달 화살표 버튼
- 드롭다운으로 년도/월 직접 선택
- 깔끔한 애니메이션

### 2. CalendarGrid (달력)

```tsx
<CalendarGrid
  year={2024}
  month={11}
  days={[
    {
      dateISO: "2024-11-05",
      summaryMinutes: 270,
      status: "completed"
    }
  ]}
  onDateClick={(date) => console.log(date)}
/>
```

- 자동 날짜 계산
- 오늘 날짜 강조
- 주말 색상 구분 (일요일: 빨강, 토요일: 파랑)
- 상태별 칩 표시

### 3. ShiftCard (근무 카드)

```tsx
<ShiftCard
  shift={{
    id: "shift-1",
    workplaceName: "스타벅스 강남점",
    startISO: "2024-11-10T09:00:00",
    endISO: "2024-11-10T17:00:00",
    status: "planned"
  }}
/>
```

- 근무지명, 날짜, 시간 표시
- 상태 배지
- 실제 근무시간 (완료된 경우)
- 메모 표시

### 달력이 제대로 표시되지 않음

2. 날짜 형식 확인 (ISO 8601: `YYYY-MM-DD`)
3. Mock 데이터 형식 확인

### 일정 추가 버튼이 작동하지 않음

현재는 라우팅만 구현되어 있습니다. 실제 일정 추가 페이지는 추후 구현 예정입니다.
